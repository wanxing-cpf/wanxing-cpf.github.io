一.Git常用命令：

操作描述 命令
工作区 → 暂存区 git add .
暂存区 → 仓库 git commit -m '提交信息'
查看状态 git status
查看分支 git branch
创建并切换分支 git checkout -b 分支名
合并分支（需先切换到目标分支） git merge 分支名
删除分支 git branch -d 分支名
强制删除 git branch -D 分支名
克隆指定仓库到本地 git clone https://github.com/你的用户名/仓库名.git
推送至远程仓库 git push origin 需推送的分支名
拉取远程最新数据 git fetch
git pull = git(fetch + merge)
